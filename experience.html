1. Research Assistance Project

This project consists of three different subprojects (all written in Python, and all within the same GitHub repository).

1. A script used for automating YouTube video data collection using the YouTube API and Gemini's API. Here's how it works: The user inputs a keyword for the script to search in YouTube. The script grabs the first 20 videos for that word, analyzes them, writes a summary and extracts valuable quotes from each video. The output is stored in a markdown file.

2. A research workflow using ExaAI's API. Built to search for tweets and store them in markdown files for easy reading. The project also has a general search agent.

3. This part of the project is still work in progress, but it will be another search Agent, which will use BrowserUse.

2. Econometrics Interactive Book

A project led by Professor Ricardo A. Pasquini, developed collaboratively with a team of students. The result is an interactive digital book designed for university students taking their first course in econometrics. It includes dynamic dashboards to help visualize and understand key concepts. I contributed specifically to the dashboard on multicollinearity. The book.

3. Defi lending analysis

This is an ongoing project focused on estimating the impact of parameter changes—whether relaxed or tightened—in DeFi lending protocols such as AAVE. My work so far has involved in-depth research, developing a solid understanding of the protocol's behavior, and, most importantly, querying and retrieving data directly from smart contracts.
